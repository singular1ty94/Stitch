<static> searchSteamStorefront(url, caller, newTitle)

This AJAX call fetches a JSON object of potential AppID's from SteamDB, using the provided name. It calls Core.getPricesAndScore if successful. It calls Core.failedPricesAndScore if SteamDB cannot find the game.
Parameters:
Name Type Description
url The url to pass the server, likely /?search=game+name
caller Which accordion link called this function?
newTitle May be undefined, the 'corrected' title of the game (see Core.failedPricesAndScore)
